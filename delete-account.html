🔮 오운완 계정 및 데이터 삭제 요청

오운완 앱에서 사용하신 계정과 관련 데이터의 삭제를 요청하실 수 있습니다.

삭제되는 데이터

계정 삭제 요청 시 다음 데이터가 영구적으로 삭제됩니다:

  • 계정 정보 — Google 로그인 연동 정보, 이메일
  • 프로필 정보 — 이름, 생년월일, 출생 시간, 성별
  • 운세 기록 — 모든 운세 조회 히스토리 및 궁합 분석 결과
  • 이용권 정보 — 무료/유료 이용권 잔여 내역
  • 기기 정보 — 푸시 알림 토큰, 기기 식별자
주의: 삭제된 데이터는 복구할 수 없습니다. 남은 이용권도 함께 삭제되며 환불되지 않습니다.

삭제 요청 방법

방법 1: 앱 내에서 삭제

  • 오운완 앱 실행 → 설정(⚙️) 탭 → "계정 삭제" 선택

방법 2: 이메일로 요청

  • 아래 버튼을 눌러 삭제 요청 이메일을 보내주세요.
  • 가입 시 사용한 Google 계정의 이메일 주소를 반드시 포함해 주세요.

처리 기간

  • 삭제 요청은 접수일로부터 7일 이내에 처리됩니다.
  • 처리 완료 시 입력하신 이메일로 안내드립니다.
